# libbash
A library of common bash functions

---

## :rocket:&nbsp; TL;DR

### :floppy_disk:&nbsp; Installation

```shell
git submodule add git@github.com:nicholaswilde/libbash 
```

### :gear:&nbsp; Usage

```shell
#!/bin/bash
source ./libbash/logging

lb_infoln "infoln test"
```
