# Development

## :robot: [Task][1] is used to automate tasks.

Task needs to be executed in the root of `libbash` repo.

## New Library

=== "Task"
    ```shell
    task new-lib LIB=mynewlib
    ```
=== "Manual"
    ```shell
    cp ./templates/lib.tmpl ./mynewlib
    ```
